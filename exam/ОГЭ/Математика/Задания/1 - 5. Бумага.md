# **Описание**

Данный раздел посвящён стандартным форматам бумаги серии А (A0, A1, A2, A3, A4, A5, A6 и т.д.), которые широко используются в России и мире. Задачи, связанные с этими форматами, часто встречаются на ОГЭ по математике. Они проверяют понимание пропорций, геометрической последовательности, вычислений площадей и отношений сторон. Все форматы подобны друг другу, что позволяет сохранять пропорции при масштабировании.

* * *

# **Теория**

![[Листы.png]]
Форматы бумаги серии А основаны на следующих принципах:

1. **Формат A0** — базовый формат, площадь которого равна **1 м²**.
2. **Соотношение сторон** любого листа формата А — **$\sqrt{2} : 1$** (приблизительно 1.414 : 1). Это обеспечивает подобие всех форматов.
3. **Получение меньших форматов**: каждый следующий формат (A1, A2, A3 и т.д.) получается делением предыдущего пополам по **большей стороне**.
4. **Площадь**: площадь каждого следующего формата в **2 раза меньше** предыдущего.
   - A0 → 1 м²
   - A1 → 0.5 м²
   - A2 → 0.25 м²
   - A3 → 0.125 м²
   - A4 → 0.0625 м²
   - A5 → 0.03125 м²
   - A6 → 0.015625 м² = **156.25 см²**

5. **Размеры сторон**:
   - Если известна площадь и соотношение сторон, можно вычислить длины сторон.
   - Для формата A0:  
     Пусть меньшая сторона = $x$, тогда большая = $x\sqrt{2}$.  
     Площадь: $x \cdot x\sqrt{2} = 1 \text{ м}^2 \rightarrow x^2\sqrt{2} = 1 \rightarrow x = \frac{1}{\sqrt[4]{2}} \approx 841 \text{ мм}$  
     → Большая сторона ≈ 1189 мм.

6. **Количество листов при разрезании**:  
   - Из одного листа формата A$n$ получается **$2^k$** листов формата A$(n+k)$.  
   - Например: из A2 → 2 листа A3 → 4 листа A4 → **8 листов A5**.
   - Количество листов, которые можно получить листа формата $A{x}$, листов формата $A{y}$  $$ K_{x->y} = 2^{x-y} $$

* * *

# **Принцип решения**

1. **Определите, что требуется**: площадь, длина стороны, количество листов, отношение сторон.
2. **Используйте соотношение сторон $\sqrt{2} : 1$** для вычисления неизвестных сторон.
3. **Площадь уменьшается в 2 раза при переходе к следующему формату**.
4. **При разрезании**: количество листов = $2^{\text{разница в индексах форматов}}$.
5. **Если даны размеры в таблице** — сопоставьте их с форматами по принципу: чем меньше индекс, тем больше размеры.
6. **При вычислении отношений** — всегда меньшая сторона делится на большую → результат < 1, округляется по требованию.

# **Примеры**

**Пример 1.**  
*Сколько листов бумаги формата A5 получится при разрезании одного листа бумаги формата A2?*  
→ Разница индексов: $5 - 2 = 3$ → $2^3 = \mathbf{8}$ листов.

**Пример 2.**  
*Найдите длину меньшей стороны листа бумаги формата A1.*  
→ A0: 841 мм × 1189 мм  
→ A1: разрезаем A0 пополам → меньшая сторона A1 = 841 мм, большая = $1189 / 2 = 594.5$ мм → **меньшая сторона = 594 мм** (округляем до мм).

**Пример 3.**  
*Найдите площадь листа бумаги формата A6.*  
→ A0 = 1 м² = 10 000 см²  
→ A6 = $10\,000 / 2^6 = 10\,000 / 64 = \mathbf{156.25}$ см²

**Пример 4.**  
*Найдите отношение длины меньшей стороны к большей у бумаги формата A2. Ответ дайте с точностью до десятых.*  
→ Соотношение сторон для всех форматов = $1 : \sqrt{2} \approx 1 : 1.414$ → отношение = $1 / 1.414 \approx \mathbf{0.7}$
